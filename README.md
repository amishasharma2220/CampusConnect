# CampusConnect — MUJ Event Management Platform

A comprehensive event management system for Manipal University Jaipur, connecting students, clubs, and faculty.

## Features

- Browse and register for campus events
- Club directory with department-wise categorization
- Academic calendar with event tracking
- Student dashboard with certificates and participation history
- Club admin panel for event creation and management
- Event approval workflow with faculty oversight

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **UI Components**: Radix UI primitives with custom theming
- **Animations**: Framer Motion
- **Routing**: React Router v6
- **Build Tool**: Vite

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Project Structure

```
│   └── ui/           # Base UI primitives
├── data/             # Mock data and type definitions

├── lib/              # Utility functions
└── pages/            # Route-level page components
```
